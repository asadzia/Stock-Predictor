# Stock Prddictor

Predicts next-day stock prices using a Random Forest model.
Uses Streamlit for UI.

# To run locally

## Install dependencies
pip install -r requirements.txt

## Train Model

python app/train.py

# Run the app
streamlit run app/main.py

# Next Steps:

Productionize.
